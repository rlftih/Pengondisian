# Pengondisian

penjelasan dan fungsi

Git Init
penjelasan: Membuat repository Git baru di dalam folder proyek.
Fungsi: Perintah ini mengaktifkan Git sehingga folder tersebut bisa mulai dilacak perubahannya.
Git status
Penjelasan:Menampilkan file mana yang sudah diubah, belum di-track, atau siap di-commit
Fungsi:Melihat kondisi repository saat ini
Git commit
Penjelasan:Perubahan yang sudah di-stage akan disimpan sebagai versi baru
Fungsi:Menyimpan perubahan ke dalam repository
Git push origin
Penjelasan: origin adalah nama default remote, jadi perintah ini meng-upload perubahan ke server.
Fungsi: Mengirim commit dari lokal ke repository remote (misalnya GitHub)
Git add 
Penjelasan: Perintah ini digunakan sebelum git commit, untuk menentukan file mana yang ingin disimpan perubahannya.
Fungsi: Menambahkan file ke staging area (area siap commit).
